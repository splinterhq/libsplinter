## `clear` CLI User's Reference

The purpose of `clear` is to clear the screen.

### Arguments & Switches

None — this command takes no arguments.

### Example Uses

**Console:**
```
splinter_debug # clear
```

**Shell:**
```
$ splinterctl clear
```

### Additional Information And Rationale

**Additional Info (Or None):**
`cls` is an alias of `clear`.

**Rationale (Or None):**
None
