## `uuid` CLI User's Reference

The purpose of `uuid` is to generate and print a UUID v4 unique identifier.

### Arguments & Switches

None — this command takes no arguments.

### Example Uses

**Console:**
```
splinter_debug # uuid
244cc1eb-baf8-41ea-beee-f634f3c00f61
```

**Shell:**
```
$ splinterctl uuid
```
